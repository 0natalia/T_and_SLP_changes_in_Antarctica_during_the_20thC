* ### wave3_index.m

Calculates the Zonal Wave 3 (ZW3) index, according to Raphael (2004), and its trend;

Map plots the spatial correlation between the ZW3 index and T/SLP fields.

* ### sam_index.m

Calculates the Southern Annular Mode (SAM) index, according to Gong and Wang (1999), and its trend;

Map plots the spatial correlation between the SAM index and T/SLP fields.
